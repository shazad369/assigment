What is JSX, and why is it used?

 ans:JSX মানে হলো JavaScript XML।
এটা মূলত React-এ ব্যবহৃত হয় যাতে আমরা JavaScript কোডের ভেতরে HTML-এর মতো সিনট্যাক্স লিখতে পারি।


What is the useState hook, and how does it work?
ans:React-এ useState একটা Hook, যেটা ব্যবহার করে আমরা কোনো component-এর ভেতরে নিজের data রাখতে পারি এবং সেটা প্রয়োজন অনুযায়ী পরিবর্তনও করতে পারি।

How can you share state between components in React?
ans:React-এ যদি একাধিক component-এর মধ্যে একই data বা state ব্যবহার করতে হয়, তাহলে সেই state টা parent component-এ তোলা হয় — একে বলে “Lifting State Up”।
তারপর parent component সেই state-টা props আকারে child component-গুলোর মধ্যে পাঠিয়ে দেয়।
